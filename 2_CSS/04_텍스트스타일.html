text-decoration

글자에 밑줄/윗선을 긋거나 취소선을 긋는 속성

underline : 글자에 밑줄을 긋는다.

overline : 글자 위에 선을 긋는다.

line-through : 취소선을 긋는다.

none : 글자에 선을 긋지 않는다.

text-transform

영문자를 표시할 때 대소문자를 원하는 대로 바꿀 수 있는 속성

uppercase : 모든 영문자 다 대문자로

LOWERCASE : 모든 영문자 다 소문자로

capitalize : 영문자의 첫글자만 대문자로
